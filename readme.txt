This is Phone Book Java project. This application helps the customer to save his phone contact and can retrieve it later at anytime. 
Customer can instert, delete and append contact. 
Customer can save phone number, facebook account, instegram account in the contact. 
Customer can search for any contact by the first name or the last name. 

Additions :- 

- Customer can save the data in file by json format or regular format as exist in the pdf (Also program should be extendable to support extra formats)
- UI, TCP/IP and console should use same method to read or edit the data 

Part #2
Customer can read/remove/list/update contact via URL by using TCP/IP in java
